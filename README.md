# Vue Community Project

这是一个基于 Vue 3 + TypeScript 的 Monorepo 社交社区项目。

## 📖 项目简介

本项目旨在打造一个现代化的社交平台，在 UI 设计上致敬 Instagram 的沉浸式体验，同时在功能上融合了“百度贴吧”的社区频道逻辑，旨在探索流式社交与圈子社交的结合。

## ✨ 核心特色

### 1. 界面风格 (Instagram Style)
- **视觉复刻**：整体 UI 风格深度参考 Instagram，追求极简、流畅的视觉体验。
- **交互体验**：实现了类似 IG 的快拍 (Story) 滚动、无限流动态展示、侧边/底部响应式导航栏。
- **响应式设计**：完美适配桌面端（宽屏/窄屏）与移动端，提供一致的用户体验。

### 2. 贴吧频道功能 (Tieba Channels)
- **兴趣圈子**：在个人关注流之外，引入了类似“贴吧”的频道（Channel）概念。
- **板块聚合**：用户可以创建或加入特定的兴趣板块，内容更加垂直和聚焦。

## 🏗️ 系统架构

本项目采用 Monorepo 架构管理，主要包含以下两个子应用：

### 🖥️ 用户前台 (community-web)
- **面向对象**：普通社区用户。
- **功能**：
  - 首页动态流 (Feed)
  - 快拍浏览 (Stories)
  - 个人主页与资料编辑
  - 频道/贴吧浏览与发帖
  - 消息通知与互动

### ⚙️ 后台管理系统 (community-admin)
- **面向对象**：平台管理员。
- **功能**：
  - 用户管理与权限控制
  - 内容审核 (帖子、评论)
  - 频道/板块管理
  - 数据统计与运营配置

## 🛠️ 技术栈

- **包管理器**: pnpm (Workspace)
- **核心框架**: Vue 3 + TypeScript
- **构建工具**: Vite
- **样式方案**: Tailwind CSS + Less
- **状态管理**: Pinia
- **工具库**: VueUse, Lodash-es, Moment.js

## 🚀 快速开始

```bash
# 安装依赖
pnpm install

# 启动用户端
pnpm dev:web

# 启动管理端
pnpm dev:admin
```
